## 파이프라인 상세 (5Phase)

5분봉이 닫힐 때마다 각 코인에 대해 에이전트 순서(s1→s2→s3→s4)로 실행됩니다.

```
on_candle_close("ETH")
         │
         ▼
    ┌─── s1 허용? (tier_1만) ─── YES ──┐
    │                                   │
    │    ┌─── s2 허용? (tier_1,2) ──┐   │
    │    │                          │   │
    │    │   ┌─── s3 (전 티어) ──┐  │   │
    │    │   │                   │  │   │
    │    │   │  ┌─── s4 ──┐     │  │   │
    ▼    ▼   ▼  ▼         ▼     ▼  ▼   ▼
    각 에이전트별 5Phase 파이프라인 실행
```

### Phase 1: SAFETY (비상 감지)

```
입력: 실시간 캔들 + 보조 데이터
         │
         ▼
┌────────────────────────────────────┐

│  2. 스프레드 폭발 (97% 이상)        │
│  3. 변동성 급증 (95% 이상)          │
│  4. 호가창 스트레스 (95% 이상)       │
│  5. 거래량 이상 (폭증/급감)          │
│  6. 펀딩율 극단 (97%/3%)           │
│  7. OI 급변 (97% 이상)             │
│  8. 청산 캐스케이드 (95% 이상)       │
│                                    │
│  심각도 합산 → 0~240               │
│                                    │
│  ┌─────────┬──────────┬──────────┐ │

│  │  leverage: 8x             │       │

## 리스크 관리 레이어

```
Layer 1: 진입 전 필터링
┌──────────────────────────────────────────┐
│  Phase 1 (Safety)   → 비상 시 거래 차단   │
│  Phase 2 (Regime)   → 시장 상태 판단      │
│  Phase 3 (Scan)     → 시그널 품질 검증    │
│  Phase 4 (Gate)     → 종합 리스크 평가    │
│  Phase 5 (Execute)  → 노출 한도 체크      │
└──────────────────────────────────────────┘

Layer 2: 포지션 관리
┌──────────────────────────────────────────┐
│  SL (스탑로스)      → ATR 기반 자동 설정  │
│  TP (테이크프로핏)  → 2단계 분할 청산     │
│  트레일링 SL        → TP1 후 자동 추적    │
│  타임아웃           → 에이전트별 최대 보유 │
│  비상 타이트닝      → Safety 악화 시 SL↓  │
└──────────────────────────────────────────┘

Layer 3: 포트폴리오 리스크
┌──────────────────────────────────────────┐
│  MDD 모드:                               │
│  ┌─────────┬──────────┬──────┬────────┐ │
│  │ NORMAL  │ CAUTION  │DEFEN │SURVIVAL│ │
│  │ <3%     │ 3~5%     │5~8%  │ 8~10%  │ │
│  │ 레버1.0x│ 레버0.7x │0.4x  │ 0.2x   │ │
│  │ 사이즈  │ 사이즈   │사이즈│ 사이즈 │ │
│  │ 1.0x    │ 0.7x     │0.5x  │ 0.3x   │ │
│  └─────────┴──────────┴──────┴────────┘ │
│                                          │
│  EMERGENCY (MDD ≥10%):                   │
│  → 전 포지션 청산 + 24시간 거래 중단      │
│                                          │
│  노출 한도:                               │
│  • 에이전트당 동시 포지션: 3개             │
│  • 포트폴리오 전체: 8개                   │
│  • 단일 코인: 자본의 30%                  │
│  • 오픈 리스크: 에이전트 자본의 3~5%       │
└──────────────────────────────────────────┘

Layer 4: AI 적응
┌──────────────────────────────────────────┐
│  Evolver (4시간):                        │
│    성적 분석 → GPT-4o → 파라미터 조정     │
│    (DNA 가중치, Gate 기준, SL 배수)       │
│                                          │
│  Market Advisor (1시간):                  │
│    시장 스냅샷 → GPT-4o → 실시간 배수     │
│    (레버리지 0.3~2.0x, SL 0.5~2.0x)      │
└──────────────────────────────────────────┘
